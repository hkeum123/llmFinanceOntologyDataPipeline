# llmFinanceOntologyDataPipeline
LLM 기반 금융 온톨로지(Ontology) 구축 및 공시 데이터 구조화 파이프라인

1. 온톨로지(Ontology) & 지식 그래프: "회사, 산업, 테마, 사건 등 복잡한 관계를 구조화"
2. 비정형의 정형화: "10-K(연간보고서) 같은 비정형 문서에서 핵심 정보를 추출해 구조화"
3. 단순함과 확장성: "불필요한 복잡도를 줄이고, 확장성을 고려해 설계"
